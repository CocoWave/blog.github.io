  4. float浮动:浮动可以实现模式转换;

    ​ 影响:浮动的影响：如果不给父盒子设置高度，当所有的子元素都浮动起来，那么父盒子没有高度。因为浮动元素不占据标准流的位置

    ​ 解决办法:1.直接给父盒子设置一个高度

    ​ 2.给父元素设置overflow：hidden；

    ​ 3.在子元素最后加一个div，并且设置clear：both——clear只能管自己，不能管别人

    ​ 4.利用伪元素来清除浮动来带的影响(给父元素加)
